# 🛍️ Galerie Interactive : De la Conception à l'Expérience Utilisateur Avancée

## 📋 Description du Projet

Ce projet est une galerie de produits moderne et interactive développée avec HTML, CSS et JavaScript pur. Il démontre l'évolution d'un simple exercice de mise en page vers une expérience utilisateur complète avec des fonctionnalités avancées d'e-commerce.

## ✨ Fonctionnalités Implémentées

### 🎨 **Design & Interface**
- **Thème adaptatif** : Basculement clair/sombre avec persistance
- **Design responsive** : Optimisé pour desktop, tablette et mobile
- **Animations fluides** : Transitions et micro-interactions élégantes
- **Gradients modernes** : Couleurs dégradées et effets visuels

### 🔍 **Système de Recherche & Filtres**
- **Recherche en temps réel** : Filtrage instantané par nom et description
- **Filtres par catégorie** : Électronique, Sport, Mode
- **Filtres par prix** : Plages de prix configurables
- **Animations de filtrage** : Apparition/disparition fluide des produits

### ❤️ **Système de Favoris**
- **Boutons interactifs** : Cœurs animés sur chaque produit
- **Persistance** : Sauvegarde dans localStorage
- **Feedback visuel** : Notifications de confirmation
- **État cohérent** : Synchronisation parfaite

### 🛒 **Panier E-commerce Complet**
- **Panier flottant** : Compteur d'articles et total en temps réel
- **Modal détaillée** : Vue complète des articles ajoutés
- **Gestion des quantités** : Boutons +/- pour chaque article
- **Persistance** : Sauvegarde automatique des données
- **Calculs automatiques** : Totaux mis à jour dynamiquement

### 📱 **Modales & Interactions**
- **Modal de détails** : Vue agrandie des produits avec informations complètes
- **Animations spectaculaires** : Effet de zoom avec particules colorées
- **Système de notation** : Étoiles et notes numériques
- **Badges produits** : Nouveau, Promotion, Populaire

### ⭐ **Système de Notation**
- **Étoiles visuelles** : ⭐⭐⭐⭐⭐
- **Notes numériques** : (4.8/5)
- **Affichage cohérent** : Sur toutes les cartes produits

## 🛠️ Technologies Utilisées

## 🎮 Comment Utiliser

1. **Rechercher** : Tapez dans la barre de recherche pour filtrer les produits
2. **Filtrer** : Utilisez les menus déroulants pour affiner les résultats
3. **Favoris** : Cliquez sur le cœur ❤️ pour ajouter/retirer des favoris
4. **Détails** : Cliquez sur "Voir détails" pour l'animation et la modal
5. **Panier** : Ajoutez des articles et gérez les quantités
6. **Thème** : Basculez entre mode clair et sombre

## 🔧 Installation & Utilisation

1. **Cloner le repository** :
   ```bash
   git clone [url-du-repo]
   cd "Codez le Vibe – De la mise en page à l'interaction avec l'IA"
   ```

2. **Ouvrir dans un navigateur** :
   - Ouvrir `index.html` directement
   - Ou utiliser un serveur local (ex: `python -m http.server 8000`)

3. **Personnaliser** :
   - Modifier les produits dans `index.html`
   - Ajuster les styles dans `styles.css`
   - Étendre les fonctionnalités dans `script.js`
